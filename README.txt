ANIMATOR README
This program is to animate objects.

Our Programs allow users to animate Rectangle and Ellipse. Every shape that is declared in
the program will have an existing time (Start Time to End Time).
A Shape features includes:
  - Name
  - Position - This is our implement of Posn class, our Class has an x and y coordinate, Noted: Position can be negative.
  - Width
  - Height
  - Color - This is a built-in awt library class.
  - Time - This is our implementation
